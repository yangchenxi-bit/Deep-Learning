# 2.模型评估与选择

```
m个样本 a个样本分类错误 错误率=Error=a/m 精度=1-Error

学习器在训练集上的误差叫训练误差/经验误差

在新样本上的误差为泛化误差

目标：让误差最小

Q：训练集上训练了一个模型 指标为Error=0 Accuracy=100%，这是不是我们要的学习器呢？
A：不是，学习器的选择更多是它在新数据上的指标情况，也就是其泛化能力。

训练集表现好 测试集表现差 过拟合 原因：学习能力过于强大 过拟合无法避免我们只能缓解
相对的还有欠拟合 欠拟合指对训练样本未学习好 原因：学习能力低下
```

## 数据集划分方法

```
注：数据划分要保证数据分布一致 即不破坏数据的分布特性 也就是第一章说的抽样

留出法：
数据集D 训练集S 测试集T D=S∪T S∩T=∅
D由S和T组成 且S和T没有交集
优点：方法、操作简单
缺点：学习器评价结果保真性

交叉验证法（k折法）：（小数据量）
1.数据集D分为k个互相互斥的子集
2.选用其中一个作为测试集，其余k-1个为训练集，得到k个模型
3.取平均值
优点：减少因样本划分带来的偏差
缺点：数据量大时模型负担极大，且每一次验证的测试集中数据少，error不准确。
```
